System Overview

This system simulates library seat occupancy behavior based on student characteristics and library management policies. It helps analyze the relationship between seat reservation behavior and library capacity utilization.

Features:

  • Simulate library environment with customizable grid size
  • Model student behavior based on personal schedules, preferences, and personality
  • Manage seat status (vacant, taken, reserved, marked for cleaning)
  • Implement time-based cleaning of unauthorized reservations
  • Support for saving and reproducing simulation data
  • Interactive command-line interface with parameter adjustment
  • Data visualization and analysis functions
